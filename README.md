# Portfolio (site statique)

Site vitrine HTML/CSS/JS (Bootstrap, animations). Aucune étape de build obligatoire.

## Prévisualiser localement

À la racine du projet :

```bash
python3 -m http.server 8080
```

Puis ouvrir [http://127.0.0.1:8080](http://127.0.0.1:8080) (ou `index.html`).

## Modifier les styles

Les sources SCSS sont dans `scss/` ; le CSS servi est principalement dans `css/`. Après modification des `.scss`, recompiler avec votre outil Sass si vous utilisez les sources SCSS directement.
